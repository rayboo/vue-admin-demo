#### 安装
    npm install

#### 启动
    npm run dev
    
#### 打包
    npm run build
#### 如果启动后无法访问，运行一下命令，安装babel,并且在跟目录添加 .babelrc 文件
    npm install babel-preset-es2015 --save-dev
    
    .babelrc 里的内容
    {
      "presets": ["es2015", "stage-2"],
      "plugins": ["transform-runtime"],
      "comments": false
    }
